## 3. DB 连接安全边界

1. AI 不保存连接串。
2. AI 不输出连接串。
3. AI 不把连接信息写入 docs。
4. AI 不把连接信息写入 Git。
5. DB 连接信息只允许来自本地环境变量、CI Secret 或人工临时输入。
6. 生产库连接必须人工明确确认。
7. 测试库 / 开发库账号必须由数据库权限控制只读范围。
