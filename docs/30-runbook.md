## 3. 标准排查命令

### 3.1 后端健康检查

```bash
curl -i http://127.0.0.1:<port>/api/health || true
```

### 3.2 端口检查

```bash
ss -lntp | grep <port> || true
```

### 3.3 前端构建检查

```bash
npm run build
```

### 3.4 Shell 脚本语法检查

```bash
find scripts -type f -name "*.sh" -print0 | xargs -0 -I{} bash -n {}
```
